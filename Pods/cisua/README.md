## Installation

cisua is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following lines to your Podfile:

```ruby
source 'https://github.com/CocoaPods/Specs.git'
source 'git@bitbucket.org:cisua/specs.git'
```

```ruby
pod 'cisua'
```
